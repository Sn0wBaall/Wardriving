
This is a Python tool to visualize data collectes whith WIGLE. It lets you filter by network type (**Wi-Fi,GSM, Bluetooth, etc.**) and display details like BSSID, SSID, capabilities and networks type in clean tables making easy to interpret directly from terminal.

| Key | Technology | Description |
| :---: | :--- | :--- |
| **W** | Wi-Fi | Standard wireless networks |
| **E** | BLE | Bluetooth Low Energy |
| **B** | Bluetooth | Bluetooth devices |
| **A** | GSM/CDMA | Another type of cellular network technology |

> [!IMPORTANT]
> If your database contains many devices, some may not be displayed due to terminal limits. Additionally, devices with an empty SSID field are automatically hidden to keep the output clean.
